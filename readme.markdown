absolute_ca.vim
=================

version: 0.1

Author : yassu

License: Apache 2.0

This plugin provides function like `<c-a>` and `<c-x>` by only using absolute value.
For example,

```
2016-01-1|1
```

where `|` is your position.
Then, when you call `absolute_ca#increment`, you have

```
2016-01-1|2
```
